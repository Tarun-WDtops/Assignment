Q.-[1] Are the HTML tags and elements the same thing?

Ans.- HTML Element is created from HTML Tags. An element has no separate existence. An Opening Tag, Closing Tag and the content between them are collectively called Element. Each HTML tag can be an HTML element

Q.-[2]What are tags and attributes in HTML?

Ans.-(a) Tags - HTML tags are keywords that tell how the website's data will be displayed on browser.
Most of HTML tags are three part : Content
(b) HTML Attributes - Html attributes are used to give additional information of a Html tags/Elements.
